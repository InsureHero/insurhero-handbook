# Módulos del Producto

Descripción detallada de los módulos principales de InsureHero.

## Módulo de Productos

### Funcionalidades

- **Creación de Productos**: Define productos de seguros con sus características
- **Paquetes**: Agrupa productos en paquetes comercializables
- **Variantes**: Crea variantes de productos con diferentes configuraciones
- **Coberturas**: Define y gestiona coberturas disponibles

### Características

- Configuración flexible de productos
- Soporte para múltiples tipos de seguros
- Gestión de precios y tarifas
- Configuración de reglas de negocio

## Módulo de Pólizas

### Funcionalidades

- **Emisión**: Emite nuevas pólizas
- **Renovación**: Gestiona renovaciones automáticas
- **Modificaciones**: Permite modificar pólizas existentes
- **Cancelación**: Procesa cancelaciones

### Características

- Números de póliza únicos
- Historial completo de cambios
- Estados de póliza (activa, cancelada, vencida)
- Integración con pagos

## Módulo de Reclamos

### Funcionalidades

- **Creación**: Crea nuevos reclamos
- **Seguimiento**: Seguimiento del estado del reclamo
- **Evaluación**: Procesa evaluaciones de reclamos
- **Pagos**: Gestiona pagos de reclamos

### Características

- Números de reclamo internos y externos
- Flujos de trabajo personalizables
- Comunicación con asegurados
- Gestión de documentos adjuntos
- Eventos y historial

## Módulo de Suscripciones

### Funcionalidades

- **Gestión**: Gestiona suscripciones activas
- **Pagos**: Procesa pagos recurrentes
- **Renovación**: Renovaciones automáticas
- **Cancelación**: Procesa cancelaciones

### Características

- Ciclos de facturación configurables
- Notificaciones automáticas
- Gestión de métodos de pago
- Historial de pagos

## Módulo de Usuarios

### Funcionalidades

- **Gestión**: Crea y gestiona usuarios
- **Roles**: Asigna roles y permisos
- **Autenticación**: Maneja autenticación y autorización
- **Perfiles**: Gestiona perfiles de usuario

### Características

- Múltiples roles y permisos
- Integración con Supabase Auth
- Gestión de sesiones
- Recuperación de contraseña

## Módulo de Canales

### Funcionalidades

- **Configuración**: Configura canales de distribución
- **Integración**: Integra con sistemas externos
- **Gestión**: Gestiona múltiples canales

### Características

- Soporte para múltiples canales
- Configuración por canal
- APIs específicas por canal
- Gestión de administradores por canal

## Módulo de Integraciones

### Funcionalidades

- **Webhooks**: Configura webhooks para eventos
- **APIs**: Expone APIs para integraciones
- **IA**: Integraciones con sistemas de IA
- **Sistemas Externos**: Conecta con sistemas externos

### Características

- Webhooks configurables
- APIs versionadas
- Autenticación segura
- Logging y monitoreo
